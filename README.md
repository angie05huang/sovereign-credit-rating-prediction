# Sovereign Credit Rating Prediction

This project reproduces and extends sovereign credit rating prediction models using macroeconomic indicators from World Bank and IMF datasets. The goal was to analyze how economic and financial variables influence sovereign credit ratings and compare traditional econometric approaches with machine learning models.

## Project Overview

The notebook includes:

* Data cleaning and preprocessing of real-world economic datasets
* Sovereign credit rating aggregation and transformation
* Exploratory data analysis and visualization
* Ordered Logit regression modeling
* Gradient Boosting and XGBoost model comparison
* Model evaluation using MAE and prediction analysis

The project was inspired by research on sovereign credit ratings and modernized using machine learning techniques.

## Models Used

### Ordered Logit Regression

Used as the baseline econometric model for ordinal sovereign credit ratings.

Implemented to compare machine learning performance against traditional statistical models.

### XGBoost

Used to improve predictive performance and feature learning on structured economic data.
